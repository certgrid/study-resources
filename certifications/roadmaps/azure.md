# Microsoft Azure Certification Roadmap

A clear path through Microsoft's Azure and role-based certifications - where to start, what comes next, and which track fits your role. Each exam links to its study cheatsheet. Always confirm current exams and objectives on Microsoft Learn, since Microsoft updates the catalog regularly.

## How Microsoft certifications are tiered

| Level | What it means | Examples |
|---|---|---|
| **Fundamentals** | Entry level, conceptual, no experience needed | AZ-900, AI-901, DP-900, SC-900, PL-900 |
| **Associate** | Role-based, hands-on skills | AZ-104, AZ-204, AI-102, DP-300, SC-200 |
| **Expert** | Advanced, usually builds on an associate | AZ-305, AZ-400, SC-100 |
| **Specialty** | Deep focus in one area | AZ-140 (Virtual Desktop) |

## Start here

If you are new to Azure, start with **[AZ-900: Azure Fundamentals](../microsoft/az-900-microsoft-azure-fundamentals/)**. It teaches cloud concepts, core services, and pricing/governance, and it makes every later exam easier. From there, pick the track that matches your role.

## Tracks in detail

### Infrastructure / Administrator
For sysadmins and cloud engineers who manage Azure resources.
- **[AZ-900](../microsoft/az-900-microsoft-azure-fundamentals/)** (Fundamentals) -> **[AZ-104: Azure Administrator](../microsoft/az-104-microsoft-azure-administrator/)** (Associate) -> **[AZ-305: Solutions Architect Expert](../microsoft/az-305-azure-solutions-architect-expert/)**
- Related: **[AZ-700: Network Engineer](../microsoft/az-700-azure-network-engineer-associate/)**, **[AZ-800](../microsoft/az-800-windows-server-hybrid-administrator/)** + **[AZ-801](../microsoft/az-801-configuring-windows-server-hybrid-advanced-services/)** (Windows Server hybrid), **[AZ-140: Virtual Desktop Specialty](../microsoft/az-140-microsoft-azure-virtual-desktop-specialty/)**

### Developer
For software engineers building on Azure.
- **[AZ-900](../microsoft/az-900-microsoft-azure-fundamentals/)** -> **[AZ-204: Azure Developer](../microsoft/az-204-azure-developer-associate/)** (Associate) -> **[AZ-400: DevOps Engineer Expert](../microsoft/az-400-azure-devops-engineer-expert/)**

### AI
For anyone working with Azure AI services and generative AI.
- **[AI-901: Azure AI Fundamentals](../microsoft/ai-901-azure-ai-fundamentals/)** (Fundamentals, replaced the retired AI-900) -> **[AI-102: Azure AI Engineer](../microsoft/ai-102-azure-ai-engineer-associate/)** (Associate)

### Data
For data professionals.
- **[DP-900: Azure Data Fundamentals](../microsoft/dp-900-azure-data-fundamentals/)** -> then by focus:
  - **[DP-100: Data Scientist](../microsoft/dp-100-azure-data-scientist-associate/)** (machine learning)
  - **[DP-300: Database Administrator](../microsoft/dp-300-azure-database-administrator-associate/)** (operational databases)
  - **[DP-700: Fabric Data Engineer](../microsoft/dp-700-microsoft-fabric-data-engineer-associate/)** (analytics engineering)

### Security
For security and identity professionals.
- **[SC-900: Security, Compliance & Identity Fundamentals](../microsoft/sc-900-security-compliance-and-identity-fundamentals/)** -> then by focus:
  - **[SC-200: Security Operations Analyst](../microsoft/sc-200-microsoft-security-operations-analyst/)**
  - **[SC-300: Identity and Access Administrator](../microsoft/sc-300-microsoft-identity-and-access-administrator/)**
  - **[SC-400: Information Protection & Compliance](../microsoft/sc-400-information-protection-and-compliance-administrator/)**
- Expert: **[SC-100: Cybersecurity Architect](../microsoft/sc-100-microsoft-cybersecurity-architect/)** (builds on a security associate)

### Power Platform
For low-code makers, analysts, and consultants.
- **[PL-900: Power Platform Fundamentals](../microsoft/pl-900-microsoft-power-platform-fundamentals/)** -> **[PL-200](../microsoft/pl-200-power-platform-functional-consultant/)** / **[PL-300: Power BI Data Analyst](../microsoft/pl-300-microsoft-power-bi-data-analyst/)** / **[PL-400: Developer](../microsoft/pl-400-power-platform-developer/)** -> **[PL-600: Solution Architect](../microsoft/pl-600-power-platform-solution-architect/)**

### Modern Work (Microsoft 365)
- **[MD-102: Endpoint Administrator](../microsoft/md-102-microsoft-endpoint-administrator/)**, **[MS-102: Microsoft 365 Administrator](../microsoft/ms-102-microsoft-365-administrator/)**

## How to choose

- **Totally new?** AZ-900 first, always. It is the cheapest way to learn the vocabulary the other exams assume.
- **In a job already?** Skip to the associate that matches your role (AZ-104 for admins, AZ-204 for devs, SC-200 for SOC analysts).
- **Fundamentals are optional prerequisites**, not required - but they make associate exams much easier.
- **Expert exams** (AZ-305, AZ-400, SC-100) assume real experience; take them after the associate in that track.
