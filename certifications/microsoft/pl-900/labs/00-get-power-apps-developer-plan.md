# Lab 00 - Get a Free Power Apps Developer Plan

Everything in these labs runs in a free developer environment. This lab sets it up safely and explains what is free and what is not, so you never get a surprise bill.

## Time Needed

15-20 minutes.

## What You Will Learn

- What the Power Apps Developer Plan includes for free
- How to sign up and get your own developer environment
- Which Power Platform features need paid or trial licenses
- Where your environment lives and how to find it again

## PL-900 Concepts Covered

| Concept               | Where you see it in the lab                            |
| --------------------- | ------------------------------------------------------ |
| Developer environment | You get a free, single-user environment with Dataverse |
| Licensing awareness   | You learn standard vs premium and trial concepts       |

| Microsoft Entra ID    | Your work or developer account signs you in            |

## Licensing and Cost Notes

| Option                          | Cost               | Good for                                                     |
| ------------------------------- | ------------------ | ------------------------------------------------------------ |
| Power Apps Developer Plan       | Free               | These labs: Dataverse, apps, flows in a personal environment |
| Microsoft 365 Developer Program | Free (if eligible) | A full test tenant if you have no work account               |
| Copilot Studio trial            | Free trial period  | The agent lab (Lab 08)                                       |
| Production licenses             | Paid               | Not needed for any lab in this pack                          |

The Developer Plan is for learning and development, not production. It requires a work or school account (or a developer tenant); personal accounts like outlook.com will not work directly.

## Steps

### 1. Check what account you have

- If you have a work or school account and your organization allows sign-ups, use it.
- If not, join the Microsoft 365 Developer Program first to get a free developer tenant, then continue with that account.

### 2. Sign up for the Developer Plan

1. Go to the Power Apps Developer Plan page (search "Power Apps Developer Plan").
2. Select the option to start free / get started.
3. Sign in with your work, school, or developer-tenant account.
4. Accept the terms and complete the short sign-up.

### 3. Open the maker portal

1. Go to `make.powerapps.com`.
2. In the top-right environment picker, look for an environment named like "Yourname's Environment (developer)".
3. Select it. This is your personal developer environment with its own Dataverse database.

### 4. Bookmark the two portals

| Portal                      | URL                               | Who uses it                  |
| --------------------------- | --------------------------------- | ---------------------------- |
| Maker portal                | make.powerapps.com                | Makers building things       |
| Power Platform admin center | admin.powerplatform.microsoft.com | Admins managing environments |

### 5. Confirm what you can build for free

In your developer environment you can create Dataverse tables, canvas and model-driven apps, and cloud flows, including premium connectors for development use. That covers Labs 01-07 and 09.

## Clean Up

Nothing to clean up. The developer environment is free and does not expire while you use it, though Microsoft can reclaim long-inactive developer environments.

## Success Criteria

You are done when you can explain:

- What the Power Apps Developer Plan gives you for free
- Why the developer environment is single-user and not for production
- The difference between the maker portal and the admin center
- Which account types can sign up for the plan
