# Lab 00 - Create an Azure Free Account Safely

This is the best first lab for anyone starting an Azure career. The goal is not just to create an account, but to create it safely so you understand billing, security, and cleanup from day one.

## Time Needed

20-30 minutes.

## What You Will Learn

- How to start with Azure
- Why billing details are required
- How to secure your account
- How to avoid surprise charges
- Where to find subscriptions and cost tools

## AZ-900 Concepts Covered

| Concept                    | Where you see it in the lab                |
| -------------------------- | ------------------------------------------ |
| Subscription               | Your Azure account contains a subscription |
| Billing                    | You review billing and cost settings       |
| Microsoft Entra ID         | Your account is tied to a tenant           |
| Multifactor authentication | You secure sign-in with MFA                |
| Cost Management            | You learn where budgets and spend live     |

## Steps

### 1. Start the signup

1. Go to the Azure free account page: <https://azure.microsoft.com/free/>
2. Sign in with a Microsoft account or work/school account.
3. Follow the identity verification steps.
4. Add billing details if requested.

Billing details are commonly required for verification and future paid usage. Always read the current Azure free account terms on Microsoft’s site before continuing.

### 2. Open the Azure portal

Go to:

```text
https://portal.azure.com
```

This is the main web interface for managing Azure resources.

### 3. Review your subscription

1. Search for **Subscriptions**.
2. Open your subscription.
3. Review the subscription name, subscription ID, and billing information.

Exam idea: a subscription is a billing, access, quota, and management boundary.

### 4. Secure the account

1. Open your account security settings.
2. Enable multifactor authentication if it is not already enabled.
3. Use a strong password.
4. Do not share your account credentials.

Career habit: secure the account before creating resources.

### 5. Find Cost Management

1. Search for **Cost Management**.
2. Open **Cost Management + Billing**.
3. Review where costs, budgets, and alerts are shown.

You do not need to create resources yet. Just learn where cost visibility lives.

## Clean Up

No Azure resources were created in this lab, so there is nothing to delete.

## Success Criteria

You are done when you can explain:

- What an Azure subscription is
- Why billing information matters
- Where to view costs
- Why MFA matters
- Why every beginner should learn cleanup before deploying resources
