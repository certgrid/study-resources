# Lab 01 - Navigate the Azure Portal

This lab helps you become comfortable with the Azure portal before creating resources.

## Time Needed

15-20 minutes.

## What You Will Learn

- How to search for services
- How favorites work
- Where subscriptions, resource groups, and Cost Management appear
- How to find service documentation from the portal

## AZ-900 Concepts Covered

| Concept         | Where you see it in the lab                 |
| --------------- | ------------------------------------------- |
| Azure portal    | Main web-based management tool              |
| Resource groups | You locate the resource group service       |
| Subscriptions   | You locate your subscription boundary       |
| Cost Management | You find cost and billing tools             |
| Cloud Shell     | You identify browser-based command-line use |

## Steps

### 1. Open the portal

Go to:

```text
https://portal.azure.com
```

### 2. Use the search bar

Search for each of these:

- Resource groups
- Subscriptions
- Virtual machines
- Storage accounts
- Microsoft Entra ID
- Cost Management
- Monitor

Do not create anything yet. Just learn how to find services.

### 3. Pin useful services

Pin commonly used services to your favorites:

- Resource groups
- Virtual machines
- Storage accounts
- Cost Management
- Microsoft Entra ID

### 4. Explore a service page

Open **Storage accounts** and look at the left menu. Notice common sections:

- Overview
- Activity log
- Access control (IAM)
- Tags
- Diagnose and solve problems

Many Azure resources share this layout.

### 5. Find Cloud Shell

Look for the Cloud Shell icon in the portal header. You will use it in a later lab.

## Clean Up

No resources were created.

## Success Criteria

You are done when you can explain:

- How to find Azure services quickly
- Where subscriptions and resource groups are located
- Where cost management is found
- What Cloud Shell is used for
