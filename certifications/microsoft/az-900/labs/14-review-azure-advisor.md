# Lab 14 - Review Azure Advisor

Goal: understand how Azure Advisor recommends improvements across cost, security, reliability, performance, and operational excellence.

## What you learn

- Advisor gives recommendations, not raw logs
- Recommendations are grouped by category
- Advisor is different from Azure Monitor

## Steps

1. Open the Azure portal.
2. Search for **Advisor**.
3. Open **Advisor**.
4. Review the recommendation categories:
   - Cost
   - Security
   - Reliability
   - Operational excellence
   - Performance
5. Open any available recommendation.
6. Read the impact, description, and suggested action.

## Advisor vs Monitor

| Tool          | Best for                             |
| ------------- | ------------------------------------ |
| Azure Advisor | Recommendations to improve resources |
| Azure Monitor | Metrics, logs, alerts, and telemetry |

## Exam memory

If the question says "best-practice recommendations", "reduce cost", "improve reliability", or "optimize resources", think Azure Advisor.

If the question says "collect logs", "metrics", or "alerts", think Azure Monitor.
