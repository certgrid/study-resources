# Lab 01 - Explore the Model Catalog

The exam expects you to identify an appropriate model based on capabilities. The fastest way to learn that is to compare real catalog entries.

## Time Needed

15 minutes.

## What You Will Learn

- What the model catalog contains
- How to read a model card: capabilities, context window, cost
- The difference between text, multimodal, image-generation, and embedding models

## AI-901 Concepts Covered

| Concept            | Where you see it in the lab                       |
| ------------------ | ------------------------------------------------- |
| Model catalog      | You browse and filter models                      |
| Model capabilities | You compare what each model accepts and produces  |
| Context window     | You compare how much text each model can consider |
| Deployment options | You see which deployment types each model offers  |

## Steps

### 1. Open the model catalog

1. In your Foundry project, open **Model catalog**.
2. Notice the providers: Microsoft, OpenAI, and other publishers.

### 2. Compare three models

Pick one general chat model, one multimodal model, and one embedding model. For each, note:

| What to record     | Why it matters on the exam                    |
| ------------------ | --------------------------------------------- |
| Input types        | Text only vs text + images/audio (multimodal) |
| Output types       | Text, images, or vectors                      |
| Context window     | How much content fits in one request          |
| Deployment options | Serverless API, managed compute, or both      |

### 3. Answer the picker questions

Using only the catalog cards, decide which of your three models you would pick for:

1. A chatbot answering questions about uploaded photos.
2. Semantic search over a document library.
3. A cheap, fast FAQ assistant.

## Clean Up

Nothing to clean up; browsing the catalog is free.

## Success Criteria

You are done when you can explain:

- What a multimodal model accepts that a text model does not
- What an embedding model returns and what it is used for
- Two things you would compare before choosing between two chat models
